# Minilibc
Mini lib c coded in x86_64 assembly

## Usage

```console
$ make help
all                            Build
clean                          Clean project
debeug                         Build with debeug symbols
debeug_mac                     Debeug for mac
fclean                         Restart to zero
help                           Display help
mac                            Build for mac
re                             Restart to zero and build
```
The all rule creates a shared object (.so)
